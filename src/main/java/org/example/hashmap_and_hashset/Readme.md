# 📘 HashMap & HashSet – Java Quick Guide

A beginner-friendly guide to understanding and using **HashMap** and **HashSet** effectively in coding interviews and problem solving.

---

## 🧠 Core Idea

### 🔹 HashSet
- Stores only **unique values**
- Does **not allow duplicates**
- No guaranteed order
- Used for **existence checking**

👉 Think: **“Have I seen this before?”**

---

### 🔹 HashMap
- Stores **key → value pairs**
- Keys are unique
- Used for **counting, mapping, indexing**

👉 Think: **“How many / where / what relation?”**
